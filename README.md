# Ionic Etherum App 

<p>App móvil creada en Ionic 5 y Angular para consultar balance y transacciones de billeteras en la blockchain de Ethereum.</p>

## Tecnologías usadas

- [x] `web3js` 
- [x] `Etherscan API` 
- [x] `Tailwindcss`
- [x] `Ionic 5`
- [x] `Angular 11`
- [x] `Café Santo Domingo`
