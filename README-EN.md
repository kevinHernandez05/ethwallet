# Ionic Etherum App 

<p>Mobile app developed with Ionic 5 and Angular framework to check balance and wallet transactions on the Ethereum blockchain.</p>

## this app was developed with: 

- [x] `web3js` 
- [x] `Etherscan API` 
- [x] `Tailwindcss`
- [x] `Ionic 5`
- [x] `Angular 11`
- [x] `Santo Domingo's Coffee`
